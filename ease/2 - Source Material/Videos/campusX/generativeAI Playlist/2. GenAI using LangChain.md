What is langchain?
- opensource framework which helps building LLM based application.
- modular components and end-to-end tools
- chats bots, QnA system, RAG, autonomous agents, and more

Core features:
1. supports all the major LLMs
2. simplifies developing LLM based applications
3. Integrations available from all major tools
4. open source/free/actively developed
5. supports all major GenAI use cases
