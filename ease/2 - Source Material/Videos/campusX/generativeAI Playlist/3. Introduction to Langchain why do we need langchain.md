*Opensource framework used for developing applications using LLMs*

### A High overview system design for a application where we upload pdf and ask question from it.
![[Pasted image 20250603094959.png]]
- First we do semantic search means we try to understand users query and then fetch the pages of the PDFs which is relevant according to the query.
- now we merge users query and the pages to make system query.
- and then we sent into the brain(LLMs) and it has to be ==contex aware text generation capablity== 

#### Semantic Search

![[Pasted image 20250603095642.png]]
flow of semantic search
1. first we convert each paragraph(green one) to a vector database let say 100 dimension
2. Now whenever the query asked we convert the query to same 100 dimension vector database then the find the similarity between each paragraph and query.
3. the most similar paragraph is used to answer the query.
