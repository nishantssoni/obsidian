- We can know about the Feature importance while training unlike KNN where we use methods like forward feature selection after training to know about the importance.
- Steps to find likelihood:
	- We have likelihood probabilities for all features.
	- Sort the probability values in descending order, for each class we will get an order of features which are important.
	- Features which have high likelihoods are the most important features in classifying a data point.
- Interpretability:
	- Based on likelihood or probability of features we can get why a data point is classified to a certain class.
