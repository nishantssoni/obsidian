# Data Analysis Process

Asking question → Data Wrangling → EDA → Drawing Conclusion → Communicating results.

---

## 1) Asking Questions

**Q: How can I ask better questions?**
- Have subject expertise.
- Gain experience.

### You have to ask these types of questions:
1. What features will contribute to my analysis?
2. What features are not important for my analysis?
3. Which features have a strong correlation?
4. Do I need data preprocessing?
5. What kind of feature manipulation or engineering is required?

---

## Step 2: Data Wrangling / Munging

- Transforming data from a raw state into another format.
- Done to make data more appropriate for downstream processes like analytics.

### Three parts:
1. Gathering Data
2. Assessing Data (High-level overview)
3. Cleaning Data


## Step 3: EDA (Exploratory Data Analysis)

- **Exploring Data:** Analyzing data using graphs and statistics.
- **Augmenting Data:** Modifying data through operations like removing outliers, merging datasets, etc.
    - *Note:* This process is also called **Feature Engineering**.

---

## Step 4: Drawing Conclusions

- We use **Machine Learning** to draw conclusions.
- We use **Inferential Statistics**.
- We use **Descriptive Statistics** (primarily used in data analysis).

---

## Step 5: Communicating Results / Data Storytelling

- We have to be **affirmative or confident** in our delivery.
- We communicate insights through reports, blogs, PPTs, etc.

---
> [!note] 
>  The fun part is that we don't need to do this linearly; we can jump from anywhere to anywhere.
