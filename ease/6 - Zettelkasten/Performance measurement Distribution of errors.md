- We want ei’s as close as 0
- If we plot PDF and CDF for errors (ei):
	- From the PDF we can get how many errors are small.
	- ideally we require 0 errors
	- from CDF we can get the percentage of data points that have errors.
- This can be used to compare two models:
	- If model M1 CDF is above M2 CDF then model M1 is better than model M2
