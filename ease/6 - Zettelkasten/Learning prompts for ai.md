# Learning prompts for ai

### for summarising notes
---

"I will provide my input in bullet points. Based on this input, craft a response that is concise, clear, and written in markdown format. Make sure to thoroughly address every point I provide while maintaining brevity and clarity. Extract as much relevant information as possible from the input to ensure comprehensive coverage. Structure the response logically, keeping it easy to read and understand. If any detail in my input is ambiguous or open-ended, prioritize clarity and provide additional context or explanations if necessary. Maintain a professional and informative tone throughout."

I will provide my input in bullet points. Based on this input, craft a response that is concise, clear, and written in markdown format. Make sure to thoroughly address every point I provide while maintaining brevity and clarity. Extract as much relevant information as possible from the input to ensure comprehensive coverage. Structure the response logically, keeping it easy to read and understand. If any detail in my input is ambiguous or open-ended, prioritize clarity and provide additional context or explanations if necessary. Maintain a professional and informative tone throughout. Use inline math $...$ for short equations in text. Use block math $$.........$$ for standalone equations.

---
Please correct the English in the notes I’m providing and format them in a nice markdown format. I want the content to remain the same but with proper grammar and clear structure. Provide everything in one well-structured paragraph."
