- Naïve Bayes on numerical features with assumption of Gaussian is called as Gaussian Naïve Bayes
- On binary features we will have Bernoulli Naïve Bayes.
	- Bernoulli Naive Bayes (BernoulliNB) is a variant of the Naive Bayes algorithm designed for binary or boolean (0/1) data. It assumes that the features follow a Bernoulli distribution, meaning each feature is either present (1) or absent (0).
	- BernoulliNB works well when features are binary, such as "word present or not" in text data.
	- It calculates probabilities based on the presence or absence of a feature.
- We can also have Multinomial Naïve Bayes when the distribution is assumed to be Multinomial.
	- **MultinomialNB** works with count-based features (e.g., frequency of words).
- **Naive Bayes has a fundamental assumption that all features are conditionally independent.**
