# Monotonic Function


A **monotonic function** is a function that preserves the order between inputs and outputs. It can either be:

- **Monotone increasing**: If $x_1 \leq x_2$, then $f(x_1) \leq f(x_2)$.
- **Monotone decreasing**: If $x_1 \leq x_2$, then $f(x_1) \geq f(x_2)$.

## Strict Monotonicity
- **Strictly increasing**: If $x_1 < x_2$, then $f(x_1) < f(x_2)$.
- **Strictly decreasing**: If $x_1 < x_2$, then $f(x_1) > f(x_2)$.

### Key Applications:
Monotonic functions are widely used in:
- Optimization
- Calculus (e.g., checking concavity)
- Analysis (e.g., proving limits and convergence)
